## 📋 Maintenance Checklist

### Adding New Images
```
1. Find NASA image ID
2. Add gallery-item div with correct data-category
3. Use format: https://images-assets.nasa.gov/image/[ID]/[ID]~large.jpg
4. Add emoji to title
5. Add category badge
6. Increment viewGalleryItem() number
7. Test filter functionality
```

### Adding New Category
```
1. Add filter button with emoji
2. Add CSS color scheme for category
3. Add images with new data-category
4. Update documentation
5. Test filtering
```

### Performance Optimization
```
✓ Keep lazy loading enabled
✓ Use ~large.jpg format (not ~orig.jpg)
✓ Maintain hardware-accelerated properties
✓ Avoid excessive DOM manipulation
✓ Test on mobile devices
```
